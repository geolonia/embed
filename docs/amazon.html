@geolonia/embed | Amazon Location Service

Examples for Embed API for Geolonia.

Load Amazon Location Service

var provider = new window.geolonia.AmazonLocationServiceMapProvider({
  cognitoIdentityPoolId: 'ap-northeast-1:d995db2c-8646-4ed2-8122-45dbd2c80019',
  mapName: 'explore.map',
})
provider.initMap({ container: '#map' })
